## Getting Started

### Prerequisites

- Node.js 18+ installed
- Supabase account and project
- Twilio account with phone number
- OpenAI API key (or Groq for free tier)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/agentflow.git
   cd agentflow
   ```

2. **Install dependencies:**
   ```bash
   npm install
   cd frontend && npm install
   cd ../websocket-server && npm install
   cd ..
   ```

3. **Set up environment variables:**

   **Frontend** (`frontend/.env.local`):
   ```env
   NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

   **WebSocket Server** (`websocket-server/.env`):
   ```env
   SUPABASE_URL=your_supabase_url
   SUPABASE_SERVICE_ROLE_KEY=your_service_role_key
   TWILIO_ACCOUNT_SID=your_twilio_account_sid
   TWILIO_AUTH_TOKEN=your_twilio_auth_token
   OPENAI_API_KEY=your_openai_api_key
   GROQ_API_KEY=your_groq_api_key
   WEBSOCKET_SERVER_PORT=8080
   BASE_URL=https://your-domain.com
   ```

4. **Set up the database:**
   - Go to your Supabase project
   - Open SQL Editor
   - Run migrations in order:
     - `database/migrations/001_initial_schema.sql`
     - `database/migrations/002_add_wallet.sql`
     - `database/migrations/003_add_sms.sql`
     - `database/migrations/004_add_service_architecture.sql`

5. **Start development servers:**
   ```bash
   # Terminal 1 - Frontend
   cd frontend
   npm run dev

   # Terminal 2 - WebSocket Server
   cd websocket-server
   npm run dev
   ```

   - Frontend: http://localhost:3000
   - WebSocket Server: http://localhost:8080

## Configuration

### Twilio Setup

1. **Purchase a phone number** with Voice and SMS capabilities

2. **Configure webhooks:**

   **Voice Webhooks:**
   - Voice URL: `https://your-server.com/api/voice/incoming` (POST)
   - Status Callback: `https://your-server.com/api/voice/status` (POST)

   **SMS Webhooks:**
   - SMS URL: `https://your-server.com/api/sms/incoming` (POST)

### Stripe Setup (Optional - for Text-to-Pay)

1. Create Stripe account
2. Get API keys
3. Add to environment variables
4. Configure webhook endpoint
